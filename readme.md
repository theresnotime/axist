## Installation

Import in your HTML:

```html
<link rel="stylesheet" href="https://unpkg.com/@theresnotime/axist2/dist/axist.min.css"/>
```

With npm:

```bash
npm i @theresnotime/axist2

# import '../node_modules/@theresnotime/axist2/dist/axist.min.css'
```
